# Example dataset

This directory contains a representative example dataset (Dinilysia patagonica) for running the finite-element acoustic-response workflow.

The example is intended to demonstrate the file structure and input files required by the solver.

Required files:

- `output.msh`
- `selected_vibration_face_ids.txt`
- `absorb_face_ids.txt`
- `target_coordinates.csv`

This example dataset corresponds to one representative specimen used to demonstrate the computational workflow. The full dataset used in the manuscript will be deposited in a public repository upon publication.
